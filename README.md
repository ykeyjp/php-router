ykey/router - PHP Router Library
====

## Requirement

1. PHP >= 7.1

## Installation

- composer.json
```json
{
  "repositories": [{
    "url": "https://github.com/ykeyjp/php-router.git",
    "type": "vcs"
  }],
  "require": {
    "ykey/router": "*"
  }
}
```
and `composer update`.
